### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/tclaude80/VacanceIA-multitask.git
cd VacanceIA-multitask
```

2. Install dependencies:
```bash
npm install
```

3. Create environment file:
```bash
cp .env.example .env
```

4. Configure your environment variables in `.env`

### Running the Application

```bash
# Development mode
npm run dev

# Production mode
npm start
```
